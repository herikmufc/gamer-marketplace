# 🎮 PROMPT PARA GEMINI - ÍCONES RETROTRADE BRASIL

## Contexto
Criar sprite sheet de ícones pixel art para marketplace de jogos retro com mascote gato laranja.

---

## PROMPT PRINCIPAL

```
Crie uma sprite sheet de ícones pixel art 8x8 (64 ícones) para um marketplace de jogos retro brasileiro.

ESPECIFICAÇÕES TÉCNICAS:
- Resolução: 2048x2048 pixels
- Grid: 8 colunas x 8 linhas
- Cada ícone: 256x256 pixels
- Padding: 0px entre ícones
- Estilo: Pixel art 16-bit/32-bit (não 8-bit simples)
- Bordas: Pretas grossas (4-6px)
- Cores vibrantes: Laranja #FF6B35, Azul #4A9EFF, Verde #4CAF50, Amarelo #F7B801, Vermelho #FF1744

MASCOTE:
- Gato laranja listrado (tabby cat)
- Olhos grandes e expressivos
- Pode usar óculos, fones, controle, etc
- Estilo kawaii/fofo mas com toque retro gaming

## REGRAS DE DESIGN:

1. **Consistência Visual**
   - Mesma paleta de cores em todos
   - Mesmo estilo de borda (4-6px preta)
   - Mesmo nível de detalhe
   - Iluminação consistente

2. **Legibilidade**
   - Reconhecível em 32x32 pixels
   - Alto contraste
   - Formas simples mas expressivas
   - Sem detalhes muito finos

3. **Identidade do Projeto**
   - Usar gato laranja quando possível
   - Tema retro gaming dos anos 80-90
   - Cores vibrantes e alegres
   - Estilo nostálgico mas moderno

4. **Técnica Pixel Art**
   - Anti-aliasing manual (não suave demais)
   - Outlines pretas marcadas
   - Sombras simples (flat ou 1 tom)
   - Highlights para dar volume

5. **Paleta de Cores Principal**
   - Laranja: #FF6B35 (primário)
   - Bege: #F5F1E8 (fundo/secundário)
   - Azul: #4A9EFF (tecnologia)
   - Verde: #4CAF50 (sucesso)
   - Amarelo: #F7B801 (destaque)
   - Vermelho: #FF1744 (alerta)
   - Preto: #2D2D2D (linhas)
   - Branco: #FFFFFF (highlights)
```
