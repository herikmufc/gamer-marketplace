═══════════════════════════════════════════════════════════
🚀 COMO RODAR
═══════════════════════════════════════════════════════════

1️⃣ Setup Inicial (uma vez):
   ./setup.sh

2️⃣ Configurar (uma vez):
   • API Key: backend/.env
   • IP Backend: mobile/src/api/client.js

3️⃣ Rodar Backend (terminal 1):
   ./start-backend.sh

4️⃣ Rodar Mobile (terminal 2):
   ./start-mobile.sh

5️⃣ No Celular:
   • Instalar "Expo Go" (Play Store)
   • Escanear QR Code


═══════════════════════════════════════════════════════════
📦 GERAR APK
═══════════════════════════════════════════════════════════

Opção 1 - EAS Build (Fácil):
  cd mobile
  npm install -g eas-cli
  eas login
  eas build --platform android --profile preview
  (Aguarde ~20 min e baixe APK)

Opção 2 - Expo Go (Teste rápido):
  npx expo start
  (Escanear QR Code com Expo Go)


═══════════════════════════════════════════════════════════
🤖 COMO FUNCIONA A IA
═══════════════════════════════════════════════════════════

Input:
  → 4-5 fotos do produto
  → Informações básicas (título, console, condição)

Processamento:
  → Claude 4.6 Vision analisa imagens
  → Detecta arranhões, amarelamento, autenticidade
  → Calcula raridade baseada em console/completude
  → Sugere preço usando fórmula:
    Preço = Base × Condição × Raridade × Completude

Output:
  → Condição: 0-100
  → Raridade: 0-100
  → Preço sugerido (mín/ideal/máx)
  → Insights e recomendações

Custo:
  → ~$0.03 por análise
  → Funciona sem API Key (valores padrão)
