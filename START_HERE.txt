🚀 COMO RODAR AGORA (5 MINUTOS):

1️⃣ Backend:
   cd backend
   mv main.py main_v1.py
   mv main_extended.py main.py
   python main.py

2️⃣ Configure IP (em outro terminal):
   nano mobile/src/api/client.js
   (Linha 8: API_URL = 'http://SEU_IP:8000')

3️⃣ Mobile:
   cd mobile
   npx expo start

4️⃣ No celular:
   - Instale "Expo Go" da Play Store
   - Escaneie QR Code
   - Pronto! 🎉
